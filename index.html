The Crystals Collector!

Rules:


    You will be given a random number at the start of the game that will become your target.
    By clicking the four crystals below you will add a specific amount to your total score. Each crystal will add a separate amount and that value changes with each game.
    You win the game by matching your total score to the random number.
    Each time the game starts, the game will change the values of each crystal.
